Tecnologias Utilizadas

  • React

    O React é uma biblioteca JavaScript criada pelo Facebook em 2011, projetada para o desenvolvimento de interfaces de usuário (UI) em aplicações web. Ele é amplamente utilizado devido à sua abordagem declarativa, eficiente e flexível, permitindo a criação de componentes reutilizáveis que facilitam a construção de interfaces complexas.

  • Expo

    Expo é uma plataforma open source que facilita o desenvolvimento, o build e o deploy de aplicativos mobile usando React Native, sem que você precise configurar manualmente o ambiente nativo do Android ou iOS.
